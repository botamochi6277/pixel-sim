# Pixel sim

Python codes to simulate color for LED strips


## Examples

```console
# Instagram https://uigradients.com/#Instagram
python scripts/create_colormap.py #833ab4 #fd1d1d #fcb045 --is_encode=true --name Instagram --save_fig=true
```
